Syntax
======

Spaces
------

.. _cpydiff_syntax_spaces:

uPy requires spaces between literal numbers and keywords, CPy doesn't
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Sample code::

    try:
        print(eval('1and 0'))
    except SyntaxError:
        print('Should have worked')
    try:
        print(eval('1or 0'))
    except SyntaxError:
        print('Should have worked')
    try:
        print(eval('1if 1else 0'))
    except SyntaxError:
        print('Should have worked')
